# h5-模板

## 安装依赖
```
npm install
or
# 推荐使用
npm install --registry https://registry.npm.taobao.org
```

## 打包

### 本地运行
```
npm run serve
```

### 开发环境
```
npm run build:dev
```

### 测试环境
```
npm run build:test
```

### 预生产环境
```
npm run build:test
```

### 生产环境
```
npm run build:prod
```

### 性能分析
```
npm run build:report
```
